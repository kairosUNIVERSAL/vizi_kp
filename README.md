# Ceiling KP Generator

Генератор коммерческих предложений для натяжных потолков.

## Стек технологий

- **Backend:** Python 3.14, FastAPI, SQLAlchemy, PostgreSQL
- **Frontend:** Vue.js 3, Pinia, Tailwind CSS
- **AI:** Anthropic Claude API
- **PDF:** ReportLab

## Быстрый старт

1. Создайте файл `.env` в корне `backend` (скопируйте из `.env.example`).
2. Запустите проект через Docker Compose:

```bash
docker-compose up --build
```

3. Frontend доступен по адресу: http://localhost:3000
4. Backend API docs: http://localhost:8000/docs
